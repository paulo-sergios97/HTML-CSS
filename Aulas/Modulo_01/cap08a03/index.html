Pricipais formatações:

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em negrito usando a tag <strong> (semântica).

Itálico / Ênfase

Nesta frase temos um termo em destaque usando a tag <i> (não semântica).

Nesta frase temos um termo em destaque usando a tag <em> (semântica).

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Usando as tags <big> e <small>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido mas não considerado. Tag <del>.

Texto inserido

Podmos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Tag <ins>.

Existe também o sublinhado com a tag <u> (não semântica).

Texto sobrescrito

Para inserir coisas do tipo: x20+3. Usando a tag <sup>.

Texto subscrito

Para inserir coisas do tipo: H2O. Usando a tag <sub>.
